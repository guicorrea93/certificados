## 📌 Descrição curta

Formação focada em **MySQL**, cobrindo desde consultas SQL e manipulação de dados até procedures e administração de banco de dados.

## 📖 Descrição completa

A formação **Oracle MySQL** da Alura oferece uma base sólida para trabalhar com **bancos de dados relacionais**, utilizando o MySQL como principal tecnologia.

Ao longo da formação, são abordados conceitos fundamentais de **SQL**, incluindo consultas, filtros, junções e agregações, além de **manipulação de dados (DML)** e criação de **procedures** para automação de rotinas no banco.

A formação também introduz práticas essenciais de **administração de banco de dados**, preparando o profissional para lidar com manutenção, organização e gestão de ambientes MySQL em cenários reais de uso corporativo.
