## 📌 Descrição curta

Formação focada em **Data Science com Python**, cobrindo fundamentos da linguagem, análise de dados e uso das principais bibliotecas do ecossistema.

## 📖 Descrição completa

A formação **Data Science com Python** da Alura oferece uma base sólida para atuação em projetos de análise e ciência de dados, utilizando **Python** como linguagem principal.

Ao longo da formação, são abordados desde os fundamentos da linguagem e estruturas de dados até o uso intensivo das bibliotecas **NumPy** e **Pandas**, amplamente adotadas no mercado. O foco está na **leitura, transformação, limpeza, manipulação e análise de dados**, preparando o profissional para lidar com bases reais e cenários comuns em projetos de dados.

A formação cobre práticas essenciais de **ETL**, análise exploratória e organização de dados, criando uma base robusta para avançar em estatística, machine learning e visualização de dados.
