## 📌 Descrição curta

Formação focada em **Business Intelligence e Data Warehouse**, cobrindo desde conceitos fundamentais até a construção de soluções analíticas completas utilizando SQL Server, OLAP, MDX e Power BI.

## 📖 Descrição completa

A formação **Business Intelligence e Data Warehouse** da Alura apresenta uma visão completa do ecossistema de BI, abordando conceitos, arquitetura e ferramentas utilizadas para transformar dados operacionais em informação estratégica.

Ao longo da formação, são explorados temas como **modelagem dimensional**, **ETL com Integration Services**, **cubos OLAP**, **consultas MDX** e **visualização de dados com Power BI**, preparando o aluno para atuar no desenvolvimento, manutenção e consumo de soluções analíticas corporativas.

A formação conecta a base conceitual do BI com a prática técnica em **SQL Server**, oferecendo uma visão clara do fluxo de dados desde a origem até a camada de visualização.
