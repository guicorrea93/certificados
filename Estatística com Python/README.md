## 📌 Descrição curta

Formação focada nos **fundamentos e aplicações práticas de Estatística**, utilizando Python para análise de dados, testes estatísticos, regressão e experimentação.

## 📖 Descrição completa

A formação **Estatística com Python** da Alura oferece uma base estatística sólida para profissionais de dados, cobrindo desde estatística descritiva e probabilidade até regressão e análise de experimentos.

O conteúdo é fortemente orientado à aplicação prática, utilizando **Python** como ferramenta principal para exploração de dados, validação de hipóteses e construção de modelos estatísticos. Ao longo da formação, são trabalhados conceitos fundamentais como distribuições, amostragem, testes estatísticos, correlação e regressão linear.

A formação prepara o aluno para interpretar dados com rigor estatístico, apoiar decisões baseadas em evidências e estruturar análises confiáveis em projetos de Data Science e Analytics.
