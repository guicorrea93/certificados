## 📌 Descrição curta

Formação focada em **Microsoft SQL Server 2022**, abordando desde os fundamentos da linguagem SQL até administração, performance e boas práticas para ambientes corporativos.

## 📖 Descrição completa

A formação **Microsoft SQL Server 2022** da Alura oferece uma visão completa sobre o uso e a administração de um dos sistemas gerenciadores de banco de dados mais utilizados no mercado corporativo.

Ao longo da formação, o aluno aprende a escrever **consultas SQL e T-SQL**, manipular dados, criar **procedures**, **funções**, administrar bancos de dados e aplicar técnicas de **otimização de performance**.

A formação prepara o profissional para atuar tanto no desenvolvimento quanto na administração de bancos de dados SQL Server, com foco em eficiência, segurança, escalabilidade e boas práticas.
