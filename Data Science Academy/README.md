# Data Science Academy

## 📌 Descrição curta
Conjunto de cursos da Data Science Academy cobrindo fundamentos de Python para Data Science, estatística e matemática aplicada, e análise multivariada de dados.

## 📖 Descrição completa
Cursos realizados na plataforma Data Science Academy, abordando desde os fundamentos da linguagem Python para análise de dados até tópicos avançados como matemática e estatística aplicada para Data Science, Machine Learning e Inteligência Artificial, além de análise multivariada de dados. Inclui certificações em português e inglês.

---

## 📚 Cursos da formação

### 1️⃣ Fundamentos de Linguagem Python para Análise de Dados e Data Science - Nível Introdutório
Curso introdutório cobrindo os conceitos básicos da linguagem Python aplicados à análise de dados.

---

### 2️⃣ Fundamentos de Linguagem Python para Análise de Dados e Data Science - Nível Básico
Continuação dos fundamentos de Python, aprofundando em estruturas de dados e manipulação para análise.

---

### 3️⃣ Fundamentos de Linguagem Python para Análise de Dados e Data Science - Nível Intermediário
Nível intermediário com foco em bibliotecas e técnicas mais avançadas de análise de dados com Python.

---

### 4️⃣ Matemática e Estatística Aplicada para DS, ML e IA
Curso de matemática e estatística aplicada para Data Science, Machine Learning e Inteligência Artificial.

---

### 5️⃣ Data Science para Análise Multivariada de Dados
Curso focado em técnicas de análise multivariada aplicadas à Ciência de Dados.
