## 📌 Descrição curta

Formação focada em **modelagem de dados relacional**, abordando desde conceitos conceituais até a implementação física de bancos de dados, garantindo estruturas consistentes, escaláveis e eficientes.

## 📖 Descrição completa

A formação **Modelagem de Dados** da Alura apresenta os fundamentos essenciais para projetar bancos de dados relacionais de forma correta e sustentável.

Ao longo da formação, são exploradas as etapas de **modelagem conceitual**, **lógica** e **física**, além de conceitos fundamentais como **entidades**, **atributos**, **relacionamentos**, **normalização** e **álgebra relacional**.

A formação prepara o profissional para estruturar dados de forma consistente, facilitando consultas eficientes, integridade da informação e manutenção dos sistemas ao longo do tempo.
